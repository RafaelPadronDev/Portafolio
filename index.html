Mi área de estudio principal es el Backend, donde estoy adquiriendo las bases para el desarrollo de lógica del servidor con Python y la gestión de la información con MySQL. También me formo en el diseño de interfaces con HTML y CSS para tener una visión completa. Mi propuesta de valor como trainee: En tu equipo, puedo aportar mis fundamentos sólidos en el desarrollo de lógica y el manejo de datos, sumado a mi capacitación autodidacta en FastAPI para construir APIs. Entiendo y utilizo Git y GitHub para un flujo de trabajo ordenado. Mi mayor fortaleza es mi actitud: soy proactivo, aprendo con rapidez, disfruto ayudando a mis compañeros y tengo la iniciativa para investigar y resolver problemas desde cero. Busco una oportunidad de prácticas profesionales donde pueda validar y expandir mi conocimiento técnico, comprometiéndome a ser un miembro valioso y en constante crecimiento.
